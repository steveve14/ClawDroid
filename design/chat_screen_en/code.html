Safe JSON Parsing

Certainly! When dealing with JSON, it's crucial to handle potential JSONDecodeError exceptions and missing keys gracefully. Here's a robust function using Python's built-in json module.

python 
import json

def extract_keys_safely(json_string, keys_to_extract):
    """
    Parses JSON and extracts specific keys safely.
    """
    try:
        data = json.loads(json_string)
        if not isinstance(data, dict):
            return {"error": "Parsed JSON is not a dictionary."}
        
        extracted = {key: data.get(key, None) for key in keys_to_extract}
        return extracted

    except json.JSONDecodeError as e:
        return {"error": f"Invalid JSON string: {e}"}

This approach ensures that even if the JSON is malformed or the keys are missing, your application won't crash. It returns `None` for missing keys by using the `.get()` method.
