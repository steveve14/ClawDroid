채널 연결 상세

Telegram Bot

텔레그램 봇을 연동하여 시스템 알림과 AI 인텔리전스를 스마트폰으로 직접 받아보세요.

연결 방법

1

텔레그램 앱을 열고 검색창에 @BotFather를 검색하여 대화를 시작합니다.

2

대화창에 /newbot 명령어를 입력하고, 안내에 따라 봇의 이름과 사용자 이름(Username)을 설정합니다.

3

생성이 완료되면 발급되는 HTTP API Token을 복사하여 아래 입력창에 붙여넣습니다.

key

API 토큰 입력

qr_code_scanner

QR 코드 스캔

모바일 기기에서 코드를 스캔하여 봇을 즉시 활성화하세요.
