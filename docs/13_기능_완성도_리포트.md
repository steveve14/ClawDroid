## 3. 미완성·리스크 항목 Top 10

1. **Room DB 암호화 부재** — 보안 H1. SQLCipher 적용 필요.
2. **OkHttp 로그 민감정보 마스킹** — 보안 H2.
3. **테스트 커버리지 ~10~15%** — 회귀 위험 상존.
4. **스킬 YAML/JSON 동적 로딩 불완전** — UI만 노출, 실제 실행 경로 미확정.
5. **Dynamic Color / 태블릿 2-pane 미지원** — 현대적 UX 미충족.
6. **PIN 브루트포스 방어 부재** — 실패 카운터/잠금 정책 없음.
7. **자동 삭제 정책 기본값 UX 미노출** — 사용자 제어 가시성 부족.
8. **Tool 인자 검증 미흡** — BrowserTool/FileTool 경로·스킴 allowlist 없음.
9. **오프라인 음성 모델/소음 제거 없음** — 환경 노이즈 취약.
10. **Crashlytics/Analytics 부재** — 운영 시 장애 탐지 어려움.

## 7. 결론

- **핵심 AI 어시스턴트 기능(채팅·음성·다채널·도구·프로바이더)은 90% 이상으로 거의 완성 단계**이며, 기능적으로는 Alpha → Beta 이행 가능 수준이다.
- **가중 평균 88.5%** 는 "기능은 완성되었으나 품질·보안 마무리 작업이 남아있다" 는 전형적 완성도 분포를 보여준다.
- 위에서 제시한 **Sprint 1~4 (~9주)** 로드맵을 이행하면 **97%+ 릴리스 레디** 상태를 달성할 수 있다.
