# 03. 기술 스택

> **최종 업데이트**: 2026-04

---

## 📱 Android 기본

| 항목 | 기술 | 설명 |
|---|---|---|
| **언어** | Java 17 | Android 공식 지원 언어 |
| **Min SDK** | API 26 (Android 8.0) | WorkManager, Foreground Service 지원 |
| **Target SDK** | API 35 (Android 15) | 최신 기능 활용 |
| **빌드 도구** | Gradle + Groovy DSL | `libs.versions.toml` Version Catalog |
| **앱 포맷** | AAB (Android App Bundle) | Play Store 배포용 |

---

## 🎨 UI 레이어

| 기술 | 버전 | 용도 |
|---|---|---|
| **XML Layouts + ViewBinding** | 안드로이드 표준 | UI 레이아웃 + 타입 세이프 바인딩 |
| **Material 3 (MDC)** | Material Components | 디자인 시스템 (Dynamic Color 지원) |
| **Navigation Component** | 최신 | Fragment 기반 화면 네비게이션 |
| **RemoteViews** | 안드로이드 표준 | App Widget |
| **Glide** | 최신 | 이미지 로딩 (프로필, 채널 아이콘) |

---

## 🏗️ 아키텍처 & DI

| 기술 | 용도 |
|---|---|
| **MVVM + Clean Architecture** | 아키텍처 패턴 |
| **멀티모듈** | `core/`, `feature/` 기반 모듈 분리 |
| **Hilt** | 의존성 주입 (Dagger 기반) |
| **RxJava 3** | 비동기 처리 및 반응형 데이터 스트림 |
| **LiveData** | ViewModel → View 데이터 바인딩 |

---

## 🤖 온디바이스 AI

| 기술 | 설명 |
|---|---|
| **Gemini Nano** | Google 온디바이스 LLM (AICore 시스템 서비스) |
| **ML Kit GenAI Prompt API** | Gemini Nano에 맞춤 프롬프트 전송 (텍스트/멀티모달) |
| **ML Kit GenAI 요약 API** | 대화 요약 (글머리 기호 형태) |
| **ML Kit GenAI 이미지 설명** | 카메라/갤러리 이미지 분석 |
| **ML Kit GenAI 음성 인식** | 온디바이스 STT (Gemini Nano 기반) |

### 지원 기기

| 기기 | 상태 |
|---|---|
| Pixel 8 Pro / Pixel 9 시리즈 | ✅ 지원 |
| Samsung Galaxy S24+ 이상 | ✅ 지원 |
| AICore 탑재 기기 | ✅ 지원 |
| 미지원 기기 | ⚠️ 클라우드 AI 폴백 |

### 폴백 전략

```
Gemini Nano (온디바이스) 
  → 사용 불가 시: 사용자 설정 클라우드 모델 
  → 미설정 시: AI 기능 비활성화 + 안내 메시지
```
